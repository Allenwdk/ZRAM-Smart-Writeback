# ZRAM 智能回写模块

## 功能亮点
✅ **精准内存计算**  
- 仅计算物理内存使用率（符合 Android 官方标准）
- 正确公式：`(MemTotal - MemAvailable) / MemTotal`

✅ **零误报事件检测**  
- 锁屏检测：三重验证（事件格式+来源进程+时间去重）
- 应用切换：智能包名提取（兼容所有 ROM）

✅ **智能日志系统**  
- 低负载：精简模式（每5秒1行）
- 高负载：自动显示关键数据
- 触发操作：记录完整上下文

## 配置参数
编辑 `service.sh` 开头的配置部分：
```bash
THROTTLE_SEC=60          # 命令节流时间（秒）
MEM_THRESHOLD_85=85      # 内存阈值1
MEM_THRESHOLD_90=90      # 内存阈值2
MEM_LOG_LEVEL=0          # 0=精简日志, 1=详细日志
